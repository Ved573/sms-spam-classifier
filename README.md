# 📩 SMS Spam Classifier

A Machine Learning web app that detects whether a message is Spam or Not Spam.

## 🚀 Live Demo

👉 https://sms-spam-classifier-xxxxx.streamlit.app

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* NLTK (NLP)
* Scikit-learn
* Streamlit

## 🔍 Features

* Text preprocessing (tokenization, stopword removal, stemming)
* TF-IDF vectorization
* Naive Bayes model
* Real-time prediction

## 📌 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
